# Understanding-Data

## Virtual Environment
### To Create a Virutual Environment
```
python -m venv venv
```  
### To activate virtual environment
```
venv\Scripts\activate
```    

## Docker Run Commands
```
docker build -t flask-ngrok-app .
docker run -p 5000:5000 flask-ngrok-app
```
